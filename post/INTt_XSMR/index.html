VSCode下配置Python开发环境

2019-01-12 vscode

  1. 安装python,设置好环境变量

  2. 打开Vscode,进入Ctrl+Shift+x进入应用商店，搜索python安装拓展.

  3. 用Vscode打开任意py源文件，按Ctrl+Shift+B,提示没有配置编译环境，点击新增，选择"others",vscode会自动创建一个tasks.json文件用来配置程序运行的参数。

  4. 修改command和args参数

    "version": "2.0.0",
    "tasks": [
        {
            "label": "python",
            "type": "shell",
            "command": "python",
            "args": [
                "${file}"
            ],
            "group": {
                "kind": "build",
                "isDefault": true
            }
        }
    ]`

  5. 再按一次Ctrl+Shift+B,程序即可运行。
