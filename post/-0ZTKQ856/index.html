Lepton使用手册

2020-02-20 工具

Lepton 是一个基于GitHub Gist的代码片段管理工具。

  • 无限数量的 公有/私有 代码片段
  • 无限数量的标签
  • 语言组
  • 支持Markdowm/Jupyter 笔记
  • 支持企业版GitHub
  • 沉浸模式
  • 可定制性
  • 亮/暗主题
  • 支持系统 macOs/Win/Linux
  • 仪表盘化
  • 可搜索
  • 可设置代理服务器
  • 免费

快捷键

功能 快捷键 备注
New Snippet Cmd/Ctrl + N 创建一个片段
Edit snippet Cmd/Ctrl + E 编辑一个片段
Submit Cmd/Ctrl + S 编辑后提交变化
Cancel Cmd/Ctrl + ESC 不保存直接退出编辑
Sync Cmd/Ctrl + R 同步远端Gist服务
Immersive Mode Cmd/Ctrl + I 打开沉浸式模式
Dashboard Cmd/Ctrl + D 打开仪表盘
About Page Cmd/Ctrl + , 打开关于页面
Search Shift + Space 打开搜索框

定制化

Lepton的外观可以通过<home_dir>/ .leptonrc !（不存在就创建一个文件）进行定制。

  • 主题（亮/暗）
  • 片段
  • 编辑器
  • 记录器
  • 代理服务器
  • 企业

技术栈

  1. 框架：Electron
  2. Bundler：WebPack,Babel,electron-builder
  3. 语言：ES6，Sass
  4. 数据库：React,Redux Thunk,Redux Form
  5. Lint : EsLint
