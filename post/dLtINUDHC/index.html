比例简化

分析

现给出支持人数A，反对人数B，以及一个上限L，请你将A比B化简为A’比B’，要求在A’和B’均不大于L且A’和B’互质（两个整数的最大公约数是1）的前提下，A’/B’≥ A/B且A’/B’- A/B的值尽可能小

整理几个关键信息点

  • 在A’和B’均不大于L且A’和B’互质

a<=L && b<=L && gcd(a,b)==1

  • A’/B’≥ A/B且A’/B’- A/B的值尽可能小

a/b>=A/B => a * B >= A * b && a/b 尽可能小

分析下来就是一道枚举题目，寻找满足条件的最小的a/b的值最小的那对a,b;变成了最值寻找问题

AC代码

#include <iostream>
#include<cstdio>
using namespace std;
/*
 * a<=L&&b<=L&&gcd(a,b)==1
 *  a/b>=A/B =》 a*B >= A*b  && a/b - A/B 尽可能小
 *  1~100 =》 L   a/b<=L   a<=L*b   b>=a/L
 *
 *  a/b <ta/tb
 *
 *  枚举
 * */

int gcd(int a,int b)
{
    while(b!=0)
    {
        int r=a%b;
        a=b;
        b=r;
    }
    return a;
}
int main(){

    int A,B,L;
    cin>>A>>B>>L;
    int y=gcd(A,B);
    A/=y;
    B/=y;
    int ta=L;
    int tb=1;
    for(int a=1;a<=L;a++)
    {
        for(int b=L;b>=a/L;b--)
        {
            if(gcd(a,b)==1 && a*B>=A*b && a*tb<=ta*b)
            {
                ta=a;
                tb=b;
                break;
            }
        }
    }
    cout<<ta<<" "<<tb;
}
